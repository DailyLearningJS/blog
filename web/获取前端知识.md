## 获取前端知识
#### （via 方应杭）
- 用 MDN 查 JavaScript 和 CSS 基础知识
- 用维基百科查编程概念
- 用 Stackoverflow 查疑难杂症
- 用 http://Caniuse.com 查兼容性
- 用 Google 搜索
- 用「site:http://github.com 关键词」搜索好用的库
- 用「关键词 MDN」来搜索 MDN
- 用 http://jsfeeds.com 获取前端最新资讯（英文）
- 遇到牛人，收藏到 GitHub
- 用 HackerNews 知道编程界最新动态
- 用知乎、微博了解国内前端界的动态
- 去 http://v2ex.com 水技术
- 尽量避开百度搜索
